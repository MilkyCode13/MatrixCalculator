Добро пожаловать в калькулятор матриц! Это самый продвинутый калькулятор матриц!*
Он позволяет выполнять все** необходимые операции над матрицами!
Чтобы вызвать справку, введите команду "help".

*  Среди всех калькуляторов матриц, созданных мной
** Все, которые требуются на 1 модуль 1 курса


Основные команды:
help        Показать эту справку.
fileformat  Описание формата файла с матрицами.
exit        Выход из программы

Команды ввода/вывода:
random      Сгенерировать случайную матрицу заданного размера и выполнить операцию
            над ней или сохранить её в памяти.
manual      Ввести матрицу заданного размера вручную и выполнить операцию над ней
            или сохранить её в памяти.
readfile    Считать матрицы из файла и сохранить их в памяти.
writefile   Сохранить матрицы из памяти в файл.
list        Вывести список матриц, сохранённых в памяти, и их размеров.
display     Вывести матрицу из памяти на экран.

Команды операций над матрицами:
trace       Подсчёт следа квадратной матрицы.
transpose   Транспонирование матрицы.
sum         Сложение двух матриц одного типа.
difference  Вычитание двух матриц одного типа.
product     Умножение двух матриц.
factorize   Умножение матрицы на число.
determinant Нахождение определителя (детерминанта) квадратной матрицы.
solvegauss  Приведение матрицы к ступенчатому или каноническому виду методом Гаусса.
            Решение СЛАУ, представленной в матричном виде.


Допустимые выражения в режиме выбора матрицы:
A       Буква — имя матрицы в памяти.
manual  Ввод матрицы из консоли.
random  Генерация случайной матрицы.
file    Чтение матриц из файла в память.
list    Вывод списка матриц в памяти в консоль.
help    Показать эту справку.


Описание формата файла:
Файл состоит из одного или нескольких блоков, каждый блок содержит описание
одной матрицы.
Формат блока:

[C:n:m]
c11 c12 … c1m
c21 c22 … c2m
 ⋮   ⋱  ⋱  ⋮
cn1 cn2 … cnm

Здесь C - буква, которой обозначена матрица (по ней можно будет обратиться
к данной матрице), n — количество строк, m — количество столбцов,
cij — элемент матрицы в i-ой строке и j-ом столбце.

Например (матрица A размера 4x5):

[A:4:5]
5.3 0 -3 2e6 -4
1 4.64 0 20001 46
-1e-5 0 -9.8 234 0
4.35 1 -5 -0.1 29.4

Строки, начинающиеся с '#', а также пустые строки не учитываются.
ВНИМАНИЕ! Разделитель целой и дробной части — точка.
